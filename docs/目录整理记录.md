## 整理原则

1. **主目录简洁** - 只保留必要的配置和入口文件
2. **功能分类** - 脚本、文档、数据分开存放
3. **隐藏临时文件** - 构建和数据文件不提交到git
4. **统一命名** - 使用清晰的目录名称
