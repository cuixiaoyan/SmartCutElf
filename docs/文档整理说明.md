## 使用指南

### 查看文档
```bash
# 查看主文档
cat README.md

# 查看docs目录
cd docs
cat README.md
```

### 快速开始
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行程序
python main.py

# 3. 打包程序
python build.py
```
