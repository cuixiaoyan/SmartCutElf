# SmartCutElf 使用说明

## 🎯 快速开始

```bash
# 1. 启动应用
python main.py

# 2. 在界面中操作
# - 点击"打开文件夹"选择视频目录
# - 选择视频文件预览
# - 点击"开始处理"自动剪辑
# - 在 output/ 目录查看结果
```

## 📖 使用流程

### 1. 打开文件夹
点击 `📂 打开文件夹` 按钮，选择包含视频的目录

### 2. 选择视频
- 左侧列表显示所有视频文件
- 点击文件查看详细信息
- 查看文件信息（大小、格式、时长等）

### 3. 配置选项
- **字幕+配音**：勾选启用AI字幕和配音功能
- **转场效果**：在设置中选择转场类型
- **主题切换**：切换深色/浅色主题

### 4. 开始处理
- 点击 `▶️ 开始处理` 开始自动剪辑
- 查看实时进度和日志
- 处理完成后自动提示

### 5. 查看结果
- 输出目录：`output/`
- 视频文件：`原文件名_edited.mp4`
- 字幕文件：`原文件名_edited.srt`（如启用）

## ⚙️ 设置说明

点击 `⚙️ 设置` 按钮可配置：

### 常规设置
- 输出文件夹路径
- 界面主题（深色/浅色）

### 剪辑设置
- **视频比例**：原始/横屏/竖屏
- **时长控制**：最小/最大目标时长
- **转场效果**：类型和时长
- **并行处理数量**

### AI 模型
- **Whisper 模型**：tiny/base/small/medium/large
- **配音音色**：女声/男声
- **背景音乐**：启用和音量
- **高光检测权重**：音频/视频权重

## 💡 使用技巧

### 首次使用建议
1. 用 1-2 分钟短视频测试
2. 先关闭字幕功能测试基本剪辑
3. 逐步启用高级功能

### 性能优化
- 减少并行处理数量（默认4）
- 使用 tiny 模型而非 base
- 关闭不需要的功能

### 最佳实践
- 视频格式：MP4 最佳
- 视频时长：10分钟内效果最好
- 输出质量：1080p 推荐

## 🐛 常见问题

### 找不到 FFmpeg
**解决**：查看 [FFmpeg安装指南](FFmpeg安装指南.md)

### 字幕功能报错
**解决**：安装 Whisper 或关闭字幕功能
```bash
pip install openai-whisper torch
```

### 处理速度慢
**解决**：
- 使用更小的模型（tiny）
- 减少并行数量
- 处理较短视频

### 内存不足
**解决**：
- 减少 `max_workers` 值
- 关闭其他应用
- 处理较小文件
