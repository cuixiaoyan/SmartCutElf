# 问题修复记录

## 1. PyQt5 DLL加载失败

**错误**：`ImportError: DLL load failed while importing QtWidgets`

**原因**：PyQt5-Qt5和PyQt5-sip依赖包缺失

**解决**：
```bash
pip uninstall PyQt5 PyQt5-Qt5 PyQt5-sip -y
pip install PyQt5==5.15.11
```

## 2. scipy模块缺失

**错误**：`ModuleNotFoundError: No module named 'scipy'`

**原因**：scipy未安装

**解决**：
```bash
pip install scipy numpy opencv-python pyyaml psutil openai-whisper
```

## 3. numpy.testing运行时错误

**错误**：`No module named 'numpy.testing'`

**原因**：build_fast.py中错误排除了numpy.testing

**解决**：保留numpy.testing模块
```python
# 不要排除
# '--exclude-module=numpy.testing',
```

## 4. scipy被错误排除

**问题**：build.py中排除了scipy主模块

**影响**：audio_analyzer.py无法使用scipy.io.wavfile

**解决**：只排除scipy.testing
```python
# 错误
'--exclude-module=scipy',

# 正确
'--exclude-module=scipy.testing',
```

## Python版本兼容性

当前使用Python 3.13.7，如遇到兼容性问题，建议降级到Python 3.11（最稳定）。
