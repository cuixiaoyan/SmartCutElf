# 打包方式精简说明

## 变更内容

### 精简前
- `build.py` - 标准打包（10-30分钟）
- `build_fast.py` - 极速打包（40秒）
- 两种打包方式并存

### 精简后
- `build.py` - 唯一打包脚本（40秒，优化版）
- 删除了 `build_fast.py`
- 统一使用优化后的打包方式

## 打包方式

### 命令行
```bash
python build.py
```

### 批处理
```bash
快速打包.bat
```

## 打包特点

- ⚡ **打包时间**：约40秒
- 📦 **输出体积**：约420MB
- 📂 **输出模式**：onedir（文件夹形式）
- 🚀 **优化策略**：
  - 排除torch（Whisper自动下载）
  - 排除20+个不需要的PyQt5模块
  - 排除测试模块
  - 保留numpy.testing（避免运行时错误）

## 如需单文件exe

修改 `build.py`，在PyInstaller参数中添加：
```python
'--onefile',
```

注意：打包时间会增加到5-10分钟。
